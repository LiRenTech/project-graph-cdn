# Project Graph 项目官方音效文件

## 使用方法

1. 点击 Code > Download ZIP
2. 下载到本地
3. 解压出文件夹
4. 将文件夹放在一个稳定的位置
5. 打开Project Graph软件
6. 点击设置 声音 “从官方音效文件夹导入全部音效”
7. 弹出文件选择框，找到刚刚保存的位置，点击确定
